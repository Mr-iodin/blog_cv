    数据化运营与数据挖掘概论⚓︎

    一.4P、4C、nPnC⚓︎

    以4P为代表的现代营销理论可以追溯到1960年出版的《基础营销》一书。

    4P指的是：

    • Product（产品）：表示注重产品功能，强调独特卖点；
    • Price（价格）：根据不同市场定位，制定不同价格策略；
    • Place（渠道）：指注重分营销的培养和营销网络的建设；
    • Promotion（促销）：指企业可以通过改变营销行为来刺激消费者，以短期行为促成消费增长。

    4P的核心是Product(产品)，因此，以4P理论为核心的营销思想的企业营销战略又可以称为“以产品为中心”的营销战略。

    但随着时代发展，商品逐渐丰富起来，市场竞争越来越剧烈，尤其是在21世界以后，消费者已经成为了商业世界的核心。

    4C指的是：

    • Customer‘s Needs and Wants（消费者的需求与愿望）;
    • Cost and Value to Satisfy Consumer's Need and Wands(消费者得到满足的成本)；
    • Convenience to Buy（用户购买方便性）；
    • Communication with Consumer(与用户沟通交流)；

    4C的核心为消费者。因此，以4C理论为核心营销思想的企业营销战略可简称为“以消费者为中心”的营销战略。

    随着大数据时代的来临，日益白热化的市场竞争，越来越严苛的营销预算，海量的数据堆积和存储等，迫使现代企业不得不寻找更合适、更可控的、更可量化的、更可预测的营销思路和方法论。于是在基本思路上融合了4P和4C理论的nPnC形式理论出现了。

    以3P3C为例，数据化运营6要素内容如下：

    • Probability（概率）：营销、营销活动以概率为核心，追求精细化和精准率；
    • Product（产品）：注重产品功能，强调产品卖点；
    • Prospects（消费者，目标用户）；
    • Creative（创意、包括文案活动等）；
    • Channel（渠道）；
    • Cost/Price（成本/价格）；

    在这其中，以数据分析挖掘所支撑的目标响应概率（Probability）是核心，在此基础上围绕产品功能优化、目标用户细分、活动（文案）创意、渠道优化、成本的调整等重要环节和要素，共同促进数据化运营持续完善，直至成功。

    PS：这里的目标响应概率（Probability）不应狭义理解为仅仅是预测响应模型之类的响应概率，它有更宽泛的含义，即可以从宏观上理解，由可以从微观上诠释。

    • 从宏观上，概率可以上特定消费群体整体上概率的可能性（比如卡方检测发现某特定群体在某个消费行为指标上具有显著性特征）；
    • 从微观上，概率可以是具体到某个特定消费者的“预期响应概率”（比如通过逻辑回归算法搭建一个预期响应模型，得到每个用户的预期响应概率）；

    2、数据化运营的主要内容

    ​ 尽管每个行业对“数据化运营”的定义有所区别，但其基本要素和核心是一致的，那就是“以企业级海量数据的存储和分析挖掘应用为核心支持的，企业全员参与的，以精准、细分和精细化为特点的企业运营制度和战略。”

    ​ 针对互联网运营部门的数据化运营，具体包括“网站流量监控分析、目标用户行为研究、网站日常更新内容编辑、网站营销策略推广”等，并且，这些内容是在以企业级海量数据为的存储、分析、挖掘和应用为核心的技术支撑的基础上，通过可量化、可细分、可预测等一系列精细化等方式来进行的。

    二、数据挖掘⚓︎

    ​ 数据挖掘是KDD（知识发现）的核心部分，它指的是从数据集合中自动抽取隐藏在数据中的那些有用信息的非平凡过程，这些信息表现形式为：规则、概念、规律及模式等。总体来说，数据挖掘融合了数据库、人工智能、机器学习、统计学、高性能计算、模式识别、神经网络、数据可视化、信息检索和空间数据分析等多个领域的理论和技术。

    2.1 统计分析和数据挖掘的区别⚓︎

    从理论来源上，两者很多情况下是同根同源的，数据挖掘中的技术有相当比例是用统计学中的多变量分析来支撑，当前更主流的观点为“数据挖掘是统计分析技术的衍生和发展”，两者不能完全被割裂开来。如果硬是要加以区别，两者在一下几方面会有比较明显的差异：

    • 是否需要对数据进行假设。统计分析基础之一是概率论，在做统计分析时分析人员常常要对数据分布和变量间关系进行假设，以此确定该用什么概率函数来描述变量之间的关系，以及如何检验参数的统计显著性；数据挖掘中分析人员不需要对数据分布进行任何假设，算法会自动找到变量间的关系，因此，对与一些大量杂乱的数据，数据 挖掘会更有优势。

    • 目标侧重点不同。统计分析在预测应用中常表现为一个或一组函数关系式。而数据挖掘在预测应用中的重点在于预测结果，很多时候不会从结果中产生明确的函数关系式，有些时候甚至不知道到底哪些变量在起作用，又是如何起作用（就比如神经网络，它隐藏层就是一个黑盒，没有人能在所以情况下读懂里面的非线性函数是如何对自变量进行组合，虽然这在很大程度上影响了模型在实践过程中的可理解性和接受程度，但咱们换种思维方式，只要它能准确预测客户行为，不了解模型的技术细节又有何不可）。

    • 两者解决问题思维方式不同。统计分析常需要技术人员做假设判断，然后用数据分析技术来验证假设是否成立。而数据挖掘不需要对数据的内在关系做假设判断，而是会让挖掘工具的算法自动寻找数据中隐藏的关系或规律。

    2.2 主要技术⚓︎

    （1）决策树

    ​ 在数据挖掘中非常成熟并被普遍采用，其主要优点在于它所产生的一系列从树根到树枝的规则，可以很容易的被分析师和业务人员理解，而且这些典型规则甚至不用整理，就是现成的业务优化策略和业务优化路径；

    ​ 同时，决策树对数据的分布甚至缺失非常宽容，不容易受极值的影响。

    （2）神经网络

    通过数学算法来模仿人脑思维，是数据挖掘中机器学习的典型代表。简单来说，神经网络就是通过输入多个非线性模型以及不同模型之间的加权互联，最终得到一个输出模型。

    神经网络具有大规模并行结构和信息并行处理等特点，因此具有很好的自适应性、自组织性和高容错行，并且具有较强的学习、记忆、识别功能。

    主要缺点就是其知识和结果的不可解释性。其结果很多时候让人看不清其中的逻辑关系。（但这个缺点并没有影响它在数据化运营中的广泛应用，甚至可以认为：正是因为它的不可解释性，才更有可能促使我们发现新的、没有认识到的规律和关系。）

    主要用途：分类、预测问题。

    （3）回归

    回归分析包括线性回归，这里主要指多元线性回归、逻辑斯蒂回归。

    多元回归主要描述一个因变量如何随一批自变量的 变化而变化，其回归公式就是因变量和自变量关系的数据反映。而因变量变化主要分为两部分：系统性变化与随机变化。（系统性变化是由自变量引起的，随机变化是不能由自变量解释的，通常也称为残值。）采用最小二乘法的参数估计方法。

    逻辑斯蒂回归相比于线性回归来说，在数据化运营中更主流更平凡的使用。简单来说，凡事预测“二选一”时间的可能性，都可以采用“逻辑斯蒂回归方程”。它预测的因变量介于0和1之间的概率，如果对这个概率进行换算，就可以用线性公式描述因变量与自变量的关系： $$ log\left ( \frac{p（y=1）}{1-p（y=1）} \right ) =\beta _0+\beta _1x_1+\beta _2x_2+...+\beta _kx_k $$ 采用最大似然的参数估计方法（原理是找到这样一个参数，可以让样本数据所包含的观察值被可能性最大）

    （4）关联规则

    主要目的是找出数据集中频繁模式，即多次重复出现的模式和并并发关系（同时出现的关系，频繁和并发的关系也称为并联）。其中最经典的就是购物篮分析，通过分析客户的购物篮中商品的关联，挖掘客户的购物习惯，比如以下简单的关联例子： $$ 婴儿尿不湿→啤酒[支持率=10\%,置信度=70\%] $$ 在所有顾客中，有10%的顾客同时购买了尿不湿和啤酒，而在所有购买了尿不湿的顾客中，占70%的人还同时购买了啤酒。

    规则X→Y的支持度是指事物全集中包含X∪Y的事物百分比。支持度主要衡量规则的有用性，如果支持度太小，说明规则只是偶发事件。

    其中最著名的就是Apriori算法：

    • 生成所有频繁项目集；
    • 从频繁项目集中生成所有可信关联规则。这里的可信关联规则指的是置信度大于最小置信度阀值的规则。

    （5）聚类

    针对几个特定业务指标，可以将观察对象的群体按照相似性的相异性进行不同群体的划分。

    聚类分析算法可以分为划分的方法、层次的方法、基于密度的方法、基于网格的方法、基于模型的方法等，前两种方法最为常用。

    划分的方法：k-means;

    层次的方法：依次让相似的数据两两合并，最后生成一颗聚类树（k-means++）；

    （6）贝叶斯分类方法

    它主要用来预测成员间关系可能性。比如给定一个观察值的相关属性来判断其属于一个特定类别的概率。 $$ P\left ( H｜X \right )=\frac{P\left ( X｜H \right )P\left ( H\right ) }{P\left ( X\right ) } $$ 在条件X下，H的后验概率。

    贝叶斯定理是朴素贝叶斯分类法的基础。如果给定数据集有M个分类类别，通过朴素贝叶斯分类方法，可以预测给定观察值是否属于具有最高后验概率的特定类别，也就是，朴素贝叶斯分类方法预测X属于类别Ci时，表示当且仅当 $$ P(C_i|X)>P(C_j|X)\quad 1≤j≤m,j\neq i $$ 此时如果最大化P(Ci|X),其P(Ci|X)最大化类Ci被称为最大化后的后验假设，根据贝叶斯定理 $$ P\left ( C_i｜X \right )=\frac{P\left ( X｜C_i \right )P\left ( C_i\right ) }{P\left ( X\right ) } $$ 可见，由于P（X）对于所有的类别是均等的，因此只需要P(X|Ci)P(Ci)取最大即可。

    为了预测一个未知样本X的类别，可对每个类别Ci估算相应的P(X|Ci)P(Ci)。样本X归属与类别Ci，当且仅当 $$ P(C_i|X)>P(C_j|X)\quad 1≤j≤m,j\neq i $$ 用途：分类问题的归类等。

    （7）支持向量机

    一种新的分类方法，以结构风险最小为原则。

    • 在线性情况下，就在原空间寻找两类样本最优分类超平面；
    • 在非线性情况下，使用一种非线性的映射，将原训练集数据映射到较高维上。

    支持向量机缺点是训练数据较大；优点是，对于相对复杂的非线性的决策边界的建模能力高度准确，并且也不太容易过拟合。

    用途：预测、分类

    （8）主成分分析

    属于传统的统计分析技术范畴。通过线性组合将多个原始变量合并成若干主成分，这样每个主成分都变成了原始变量的线性组合。转变的目的：一方面可以大幅度降低原始数据的维度，同时在此过程中发现原始数据属性间的关系。

    步骤：

    • 各变量的标准化；
    • 协方差阵或者相关阵计算特征根即对应的特征向量；
    • 计算方差贡献率，并根据方差贡献率的阀值选取合适的主成分个数；
    • 根据主成分荷载大小对选择的主成分进行命名；
    • 根据主成分荷载计算各主成分的得分；将主成分进行推广和延伸即成为因子分析，因子分析在综合原始变量信息的基础上将会力图构筑若干个意义较为明确的公因子。

    （9）假设检验

    是统计学基础和核心之一，主要研究在一定条件下，总体是否具备某些特定特征。

    基本原理就是小概率事件原理，即观察小概率事件在假设成立的情况下是否发生。
